# KT06 — Cloud Databases

## Цель
Сформулировать и выполнить задания КТ: Cloud Databases.

## Содержание
- `report.ipynb` — ноутбук с кодом/диаграммами и выводами.
- `src/` — дополнительные скрипты/модули.
- `screenshots/` — скриншоты, подтверждающие выполнение.
- `data/` — при необходимости (данные большие в гит **не коммитить**).

## Как воспроизвести
```bash
python -m venv .venv
# Windows: .venv\Scripts\activate
# Linux/Mac:
source .venv/bin/activate

pip install -r ../../requirements.txt
jupyter notebook report.ipynb
```
